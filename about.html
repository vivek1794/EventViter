About EventViter

EventViter is a Hybrid-Application aiming to produce the awesome features right to your hand.


Want to know when is your close friend's wedding? Or may be his daughter's birthday party? Perhaps a meeting invite from your PM requesting your precense? This app does it all!


This app is currently being developed and you can see a lot of promising features coming up in the future versions. Check out the changelog for more details.

What's New in v2.0?
  • Added custom error page and support for v3 and v4.0.x webview browsers.
  • Made the application dynamic.
  • Data are stored in the MySQL Remote Database
  • Ability to add events dynamically through web interface (Mobile interface - Coming Soon)
  • Communication with the web server through JSON objects and jQuery.
